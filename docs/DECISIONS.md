# CartPick 확정 결정 사항

> 2026-04-13 Phase 0 확정

## 기술/인프라 결정

| # | 항목 | 결정 |
|---|------|------|
| 1 | 카카오 OAuth | 앱 등록 진행, 구현은 Phase 3 |
| 2 | 도메인/SSL | MVP는 Railway 기본 도메인, 커스텀 도메인은 배포 직전 |
| 3 | 관리자 초기 계정 | 환경변수 기반 seed (`ADMIN_EMAIL`, `ADMIN_PASSWORD`) |
| 4 | 이미지 업로드 | MVP는 관리자만, jpg/png/webp, 최대 5MB |
| 5 | 타임존 | UTC 저장 + ISO8601 응답, 클라이언트에서 KST 변환 |
| 6 | DB 진실의 원천 | **Prisma schema** (schema.sql은 참고용) |
| 7 | 관리자 비밀번호 | argon2id |
| 8 | Rate limiting | 리뷰 작성 엔드포인트 1분당 3회 |
| 9 | CORS | 환경변수 기반 origin 관리 |
| 10 | 검색 최소 길이 | 2글자 이상 |
